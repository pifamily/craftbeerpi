Craftbeerpi
=========

Ansible role to deploy Craftbeerpi 4 on a K3s cluster

Requirements
------------

K3s cluster and an ESP controlling the hardware.

Role Variables
--------------

manifest_folder: Folder where manifests are stored
cbpi_namespace: craftbeerPi Namespace on K3s


Dependencies
------------

mqtt
